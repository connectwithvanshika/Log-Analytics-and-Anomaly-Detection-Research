| #  | Anomaly Category                    | What It Means                                              | Examples                                           | Important Features                                          |
| -- | ----------------------------------- | ---------------------------------------------------------- | -------------------------------------------------- | ----------------------------------------------------------- |
| 1  | **Infrastructure Anomaly**          | Abnormal behavior in servers, VMs, containers, or hardware | CPU spike, memory leak, disk full, server crash    | CPU %, Memory %, Disk Usage, Network Traffic, Restart Count |
| 2  | **Application Performance Anomaly** | Application performance deviates from normal behavior      | API slowdown, timeout errors, latency spike        | Response Time, Latency, Throughput, Request Count           |
| 3  | **Log Anomaly**                     | Unusual log patterns or error behavior                     | Error bursts, unseen exceptions, fatal logs        | Log Level, Error Count, Exception Type, Log Frequency       |
| 4  | **Security Anomaly**                | Suspicious activity indicating threats or attacks          | Brute force attacks, unauthorized access, DDoS     | Failed Logins, IP Address, Geo Location, Access Frequency   |
| 5  | **User Behavior Anomaly**           | User actions differ significantly from normal patterns     | Mass downloads, unusual login times, account abuse | Session Duration, Click Count, Login Time, Download Count   |
| 6  | **Database Anomaly**                | Abnormal database performance or availability              | Deadlocks, replication lag, slow queries           | Query Time, Active Connections, Lock Wait Time              |
| 7  | **Network Anomaly**                 | Unexpected network behavior                                | Packet loss, bandwidth spikes, DNS failures        | Packet Loss, Latency, Bandwidth Usage, Connection Count     |
| 8  | **Business Anomaly**                | Unexpected changes in business KPIs                        | Revenue drop, payment failures, order decline      | Revenue, Orders, Conversion Rate, Refund Rate               |
| 9  | **Cloud Resource Anomaly**          | Abnormal cloud infrastructure behavior                     | Pod crashes, autoscaling failures, cost spikes     | Pod Count, CPU Requests, Cloud Cost, Resource Utilization   |
| 10 | **Distributed Trace Anomaly**       | Unusual behavior in microservice communication             | Slow service chain, dependency failures            | Trace Duration, Span Time, Service Dependencies             |
| 11 | **Time-Series Point Anomaly**       | Single abnormal data point                                 | Sudden CPU spike, isolated error                   | Value, Z-Score, Deviation from Mean                         |
| 12 | **Time-Series Contextual Anomaly**  | Data is abnormal only in a specific context                | High traffic at midnight                           | Time, Seasonality, Context Variables                        |
| 13 | **Time-Series Collective Anomaly**  | Sequence of values forms an abnormal pattern               | Continuous degradation, repeated failures          | Trend, Window Statistics, Pattern Similarity                |
| 14 | **Dependency Anomaly**              | Failure propagates through connected services              | Cache failure causing API failures                 | Dependency Graph, Service Relationships                     |
| 15 | **Configuration Anomaly**           | Unexpected behavior due to config changes                  | Wrong deployment config, feature flag issue        | Config Version, Deployment Timestamp                        |
| 16 | **Deployment Anomaly**              | Issues introduced after releases                           | Error spike after deployment                       | Release Version, Deployment Time                            |
| 17 | **Resource Utilization Anomaly**    | Abnormal resource consumption                              | Memory exhaustion, thread explosion                | CPU, Memory, Threads, File Handles                          |
| 18 | **Availability Anomaly**            | Service becomes partially or fully unavailable             | Service downtime, endpoint failures                | Uptime %, Health Checks, Error Rate                         |
| 19 | **Data Quality Anomaly**            | Unexpected issues in incoming data                         | Missing values, duplicate records                  | Null %, Duplicate Count, Schema Violations                  |
| 20 | **Root Cause Correlation Anomaly**  | Multiple symptoms linked to one underlying issue           | Database outage causing failures everywhere        | Event Correlation Score, Dependency Impact                  |
